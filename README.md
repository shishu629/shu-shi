# Shu Shi Academic Homepage

This is a static GitHub Pages-ready academic homepage generated from the provided CV and designed in a style similar to the reference personal academic page.

## Files

- `index.html` - main homepage content
- `styles.css` - responsive layout and visual style
- `script.js` - mobile navigation and current-year footer
- `assets/profile.png` - profile photo extracted from the CV

## How to publish on GitHub Pages

1. Create a new GitHub repository named `<your-github-username>.github.io`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` branch and `/root`, then save.
6. Your homepage should appear at `https://<your-github-username>.github.io/` after a short build time.

## Important things to verify before public release

- Replace or confirm the Google Scholar and ORCID links in `index.html`.
- Check publication years, volume/issue numbers, and DOIs, especially newly published or in-press papers.
- Phone numbers from the CV were intentionally not placed on the public page for privacy. Add them only if you want them public.
- If you want to provide a downloadable CV, add a public-safe PDF to `assets/` and insert a link in the `CV` section.
