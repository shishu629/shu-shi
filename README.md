# Shu Shi Academic Homepage

This version restyles the existing homepage to match the supplied academic-site reference:

- fixed white navigation bar
- light animated network background
- centered white content card
- compact blue section headings and publication lists
- responsive mobile navigation
- Technical Skills section removed

## Deploy

Upload these four files to the root of the `shishu629/shu-shi` repository:

- `index.html`
- `styles.css`
- `script.js`
- `Shi-Shu-CV.pdf`

Commit the changes to the branch currently used by GitHub Pages. The public page will update automatically after the Pages workflow completes.

The circular `SS / 石澍` monogram is used because the repository does not currently contain a portrait image.
