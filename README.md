# Shu Shi Academic Homepage

This is a GitHub Pages academic homepage for Dr. Shu Shi.

## How to deploy

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Upload all files in this folder to the root of the repository.
3. In GitHub, go to **Settings → Pages**.
4. Select **Deploy from a branch**, choose `main`, and set the folder to `/root`.
5. Visit `https://<your-github-username>.github.io` after GitHub finishes deployment.

## How to update

- Edit `index.html` to update text, publications, awards, patents, and contact information.
- Replace `assets/profile.jpg` with a new photo using the same filename to update the portrait.
- Edit `styles.css` to adjust layout, colors, spacing, and fonts.
